# 金梓才观点库

## 核心原则

1. 先查证据，再下结论。
2. 区分本人原话、原文概括、方法推演与待核实事实。
3. 引用须保留原意，并给出来源、发布日期及链接或本地来源记录。
4. 没有证据时直说“当前资料中未见”，不要凭印象补写。
5. 持仓、业绩、规模及任职信息须标明报告期和数据日期。
6. 所有输出仅供研究与学习，不构成投资建议。

## 工作流

1. 判断问题属于直接观点、方法框架、观点演变、言行对照或风格化写作。
2. 检查 `references/` 是否有对应原始材料与来源记录。
3. 优先使用一手材料；二手报道只能作线索或补充，并标注性质。
4. 多条材料按时间排序，说明观点延续、调整或证据不足。
5. 输出前核对引文、人物归属、日期、报告期及事实与推演边界。

## 当前限制

- 尚未收录金梓才语料、方法框架或基金数据。
- 不得声称已经覆盖其全部公开观点。
- 不得生成可能被误认作真实引文的仿写。
- 所需资料尚未收录时，说明缺口并等待后续阶段。

## 资源

- `references/`：后续存放来源索引、原始材料、方法分析和数据说明。
- `scripts/`：后续存放可重复、可验证的检索、索引和校验脚本。
- `docs/reference-audit.md`：参考仓库审计，不是金梓才事实来源。
- `PLAN.md`：分阶段计划与完成标准。
