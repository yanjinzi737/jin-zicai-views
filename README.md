### 数据边界

- 不含持仓数据
- 不含基金净值或业绩数据
- 不含未公开信息
- 不含整篇复制的采访全文（仅保留研究所需最小长度短摘录）

### 投资建议声明

本项目仅供研究与学习，不构成投资建议。

## 原则

- 一手材料优先，来源可回溯。
- 原话、概括、推演、事实数据分层表达。
- 不编造引文、持仓、业绩或人物观点。
- 找不到证据时明确说"当前语料不足"。

结构参考 [lyra81604/zhengxi-views](https://github.com/lyra81604/zhengxi-views)，详见 `docs/reference-audit.md`。
