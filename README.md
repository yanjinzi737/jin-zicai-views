## 快速使用

```bash
# 搜索所有数据层
python scripts/query.py search "AI算力"

# 查看方法论声明
python scripts/query.py claim --type method_statement

# 查看方法规则
python scripts/query.py method

# 查看基金持仓概况
python scripts/query.py fund

# 生成研究报告
python scripts/export_report.py method --save

# 生成统计数据
python scripts/generate_stats.py
```

## 数据边界

- 不含持仓数据（除法定报告公开的前十大重仓）
- 不含基金净值或业绩数据（仅供研究参考的 NAV 除外）
- 不含未公开信息
- 不含整篇复制的采访全文（仅保留研究所需最小长度短摘录）
