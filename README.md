# jin-zicai-views

金梓才公开观点的可溯源研究 Skill。目标是基于公开原始材料回答其如何看待行业、公司或市场阶段，并区分原话、概括、方法推演与待核实事实。

## 当前状态

项目处于基础搭建阶段。已完成参考仓库结构与 LICENSE 审计，尚未批量收集语料、基金数据，也未形成投资方法结论。

## 原则

- 一手材料优先，来源可回溯。
- 先建立索引与证据规范，再扩充语料。
- 原话、概括、推演、事实数据分层表达。
- 不编造引文、持仓、业绩或人物观点。
- 研究与学习用途，不构成投资建议。

结构参考 [lyra81604/zhengxi-views](https://github.com/lyra81604/zhengxi-views)，详见 `docs/reference-audit.md`。未复制其第三方语料或基金数据。
