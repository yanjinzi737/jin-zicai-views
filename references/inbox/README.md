# references/inbox/

此目录用于人工通过浏览器保存的待核验原始资料。

## 接受格式

PDF、TXT、MD、HTML。

## 重要规则

1. 文件进入此目录不代表已经 `verified`。
2. 必须完成正文、说话人、日期、来源和定位核验后，才能写入正式 `references/corpus/files/`。
3. 不得将受版权保护的媒体全文直接提交到公开仓库。正式语料只保存：
   - 必要短摘录（研究所需最小长度）
   - 结构化摘要
   - 来源元数据（标题、作者、日期、URL、页码/段落定位）
4. 核验完成后，原始文件应从本目录移除或确认不进入版本控制。
5. 本目录中的 PDF 和 HTML 默认被 `.gitignore` 排除，仅此 README.md 被 Git 跟踪。
