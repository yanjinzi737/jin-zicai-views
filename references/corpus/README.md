# 原始语料库 MVP

本目录保存金梓才公开资料的结构化索引。目前有 12 条候选来源记录：6 份财通基金官方法定报告、5 份权威媒体原站采访候选、1 份待追溯原始刊载方的转载采访。

重要：当前记录均为 `metadata_only` 或 `pending`，尚无一条完成正文、上下文、署名、页码与摘录的全链路核验。因此它们不能被计为“已验证观点语料”，也不能据此确认稳定方法论。

## 文件

- `index.jsonl`：规范真源，一行一条来源。
- `index.csv`：由 `scripts/build_corpus_index.py` 生成的表格视图。
- `source-map.md`：来源等级、状态和缺口说明。
- `../timeline.md`：仅基于现有证据构建的初步时间线。
- `../method.md`：候选方法论证据表。

## 状态

- `verified`：正文、来源、日期、发言归属和摘录上下文均已核验。
- `metadata_only`：标题、日期或官方链接已核验，但正文证据链未完成。
- `pending`：原始出处、转载关系或正文仍待确认。
- `restricted`：遇到合法访问限制，未尝试绕过。
- `failed`：链接或内容校验失败。

## 收录规则

法定报告优先保存官方 PDF URL；只有下载并核对基金经理、报告期、章节和页码后，才填写观点摘录并提升为 `verified`。媒体采访只保存研究需要的短摘录和结构化摘要，不整篇复制。
